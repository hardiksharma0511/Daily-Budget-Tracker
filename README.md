# 💰 Daily Budget Tracker

A clean, minimalist, and beginner-friendly budget management application built with **React** and **Vite**. This tracker helps users manage their daily income and expenses with zero math confusion, starting from a clean slate every session.

🔗 **Live Demo:** [https://hardiksharma0511.github.io/Daily-Budget-Tracker/](https://hardiksharma0511.github.io/Daily-Budget-Tracker/)

---

## ✨ Features

*   **Clean Slate Design:** Starts with an empty transaction history for a focused daily budget window.
*   **Intuitive Transaction Flow:** Simple "Add Income" and "Add Expense" actions that process calculations instantly behind the scenes.
*   **Real-time Summary:** Displays clear totals for total income, total expenses, and the remaining net balance.
*   **Soft Visual Palette:** Designed with a calming, readable interface to eliminate financial data stress.

---

## 🚀 Tech Stack

*   **Frontend Library:** React (Functional Components, Hooks)
*   **Build Tool & Dev Server:** Vite (Optimized production builds)
*   **Deployment System:** GitHub Actions & GitHub Pages
