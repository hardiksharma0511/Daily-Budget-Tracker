# 💰 Daily Budget Tracker

A modern and beginner-friendly **React** application for tracking daily income and expenses. Built to practice **React Fundamentals**, including Components, Hooks, State Management, Event Handling, and Conditional Rendering, while providing a clean and easy-to-use budgeting experience.

🌐 **Live Demo:**  
https://hardiksharma0511.github.io/Daily-Budget-Tracker/

---

## ✨ Features

- 📊 Track income and expenses in real time
- 💰 Automatically calculate current balance
- ➕ Separate **Add Income** and **Add Expense** actions for a confusion-free experience
- 🗑️ Delete transactions instantly
- 🎨 Clean, modern UI with soft colors
- 📱 Responsive design for desktop and mobile
- ⚡ Fast performance using Vite

## 🛠️ Installation

Clone the repository

```bash
git clone https://github.com/hardiksharma0511/Daily-Budget-Tracker.git
```

Go to the project directory

```bash
cd Daily-Budget-Tracker
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
